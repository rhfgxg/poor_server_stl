# 游戏段位
这里使用最古老的版本
段位分为：25-1级，传说
星级：
25-20：每级3星，失败不掉星
20-10：每级4星，失败掉星
10-1：每级5星，失败掉星
传说：不计星，进行全服隐藏分排名
连胜：
连胜3场：额外+1星
连胜4场：额外+2星
以此类推
保段：不设置保段机制
赛季结算：每个月为一个赛季，每月1日凌晨4点进行结算
不进行赛季计数
段位继承：
降四级，保留星数
传说-4级0星
1级0星-5级0星
10级0星-14级0星
14级-18级

# 游戏回合
## 对局阶段

进入回合循环
